# Order Data

Orders are generated by `scripts/08-generate-orders.sh`.

## Order Generation

The script creates:
- Orders with various statuses
- Multiple items per order
- Realistic order dates (past 6 months)
- Customer associations
- Order totals

## Order Statuses

- Completed (majority)
- Processing
- On-hold
- Cancelled

## Customization

Edit `scripts/08-generate-orders.sh` to modify order generation logic.
